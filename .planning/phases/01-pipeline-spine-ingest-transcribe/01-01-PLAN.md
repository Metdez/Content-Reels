# Plan 01-01: Pipeline Spine — Ingest + Transcribe

## Goal
CLI `content-machine ingest <video>` → `data/<job_id>/` with `source` ref, `audio.wav` (16kHz mono), `transcript.json` (segments + words), `job.json` manifest. whisper.cpp (Metal) transcribes; ffmpeg silencedetect VAD suppresses hallucinations; re-runs reuse cached transcript by content hash.

## Tasks
- **Scaffold**: pyproject.toml, package `content_machine/`, .gitignore, Makefile, scripts/setup.sh
- **config.py**: resolve data dir, vendored whisper-cli + model paths, defaults, env overrides
- **jobs.py**: `compute_job_id` (sha256 of bytes), per-job dir layout, `job.json` manifest with per-stage status, load/save/update helpers
- **transcribe.py**: ffmpeg audio extract (16k mono) → whisper.cpp run → parse full JSON → VAD filter (ffmpeg silencedetect drops segments inside silence) → write `transcript.json`; cache by hash
- **cli.py**: typer `ingest` command wiring the above with progress prints
- **tests**: job_id determinism, manifest round-trip, ffmpeg/whisper command construction, VAD filter logic, hallucination blocklist

## Artifacts this phase produces
`content_machine/{config,jobs,transcribe,cli}.py`, `compute_job_id()`, `Job`, `transcribe()`, `extract_audio()`, `build_whisper_cmd()`, `parse_whisper_json()`, `vad_filter()`, CLI `ingest`.

## must_haves (truths)
- Running `ingest` on a real mp4/mov yields `data/<job_id>/{source,audio.wav,transcript.json,job.json}`
- `transcript.json` has segment-level text + word timing, anchored to audio (verified on known-content `say`-generated test video)
- Silent clip → no phantom text (VAD)
- Re-run reuses cached transcript (no re-transcribe) keyed by video hash
